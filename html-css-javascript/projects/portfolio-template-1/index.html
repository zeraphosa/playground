.NET Core nedir?

.Net Core Microsoft tarafından Github üzerinde, kaynak kodları açık bir şekilde MIT ve Apache 2 lisanslarına sahip olarak yer almaktadır. Cross platform olarak çalışan (Windows dışındaki diğer işletim sistemlerinde de uygulamanızı çalıştırabilir), esnek ve modern geliştirme platformudur.
